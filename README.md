## üöÄ Project Goal
The goal of this project is to demonstrate how core data analysis tasks translate across the three most essential tools in the industry. It serves as a interactive handbook for:
* **Excel:** For quick prototyping and business communication.
* **SQL:** For database querying and data extraction.
* **Python (Pandas):** For advanced data manipulation, automation, and statistical analysis.

## üìÇ Repository Contents
The main analysis is located in the **[`The_Analyst_Handbook.ipynb`](The_Analyst_Handbook.ipynb)** file. It covers a complete End-to-End analysis lifecycle:

| Part | Topic | Key Skills Demonstrated |
| :--- | :--- | :--- |
| **0** | **Data Generation** | Python scripting to create realistic E-Commerce datasets. |
